### Web Version
```bash
git clone https://github.com/bigduck/pomodoro.git
cd pomodoro
open index.html

# Install dependencies
npm install

# Run development version
npm start

# Build for your OS
npm run build-win  # Windows
npm run build-mac  # macOS
